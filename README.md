# Chat Mute Skript

**Another Chat mute skript**

*Read Bellow*

This is a Useful chat mute skript to mute your chat when you need to! :)
It is all controled with commands!

```You can choose a time to mute the chat or just mute it till its unmuted | /mutechat [<time>]```

```This skript uses the following Permissions: chat.mute, chat.mute.bypass```

**Supports:**

> Latest version of skript

> Supports Bypass permissions

> Supports 1.16.1 - 1.20.2
